% 标量场论的单圈修正
% keys 重整化|单圈修正

为了消除 $\phi^4$ 理论中圈图的紫外发散，我们引入重整化后的拉氏量：
\begin{equation}
\begin{aligned}
\mathcal{L}
&=\frac{1}{2}Z_\phi \partial_\mu \phi \partial^\mu \phi - \frac{1}{2} Z_m m^2\phi^2 - \frac{Z_\lambda \lambda}{4!}\phi^4\\
&=\frac{1}{2}(\partial_\mu\phi)^2 - \frac{1}{2}m^2\phi^2 - \frac{\lambda}{4!}\phi^4
\end{aligned}
\end{equation}
