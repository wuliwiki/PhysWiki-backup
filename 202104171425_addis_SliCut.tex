% SLISC 容器的切割

如果我们想把矩阵的一行输入到某个函数中怎么办呢？ 如果我们不使用任何容器而是用指针， 函数的输入输出也使用指针， 那么可以通过 \verb|step| 参数实现：
\begin{lstlisting}[language=cpp]
Doub sum(const Doub *x, Long_I N, Long_I step)
{
    Doub s = x[0];
    for (Long i = 1; i < N; ++i)
        s += x[step*i];
    return s;
}
\end{lstlisting}

\begin{lstlisting}[language=cpp]
Long N1 = 10, N2 = 10, row = 3;
Doub *a;
new Doub a[N1*N2]; // 列主序
// 给 a 赋值
cout << "sum = " << sum(a[3])
delete a[];
\end{lstlisting}
