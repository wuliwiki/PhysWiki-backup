\begin{itemize}
\item \verb`PyObject *` 可以容纳任何 python 对象
\item \verb`PyObject *pName = PyUnicode_DecodeFSDefault("my_python_module");` 可以创建一个 python 字符串变量
\item \verb`PyObject *pModule = PyImport_Import(pName);` 可以加载 python 模块
\end{itemize}
