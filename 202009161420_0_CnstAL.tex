\pentry{速度 加速度（一维）\upref{VnA1}}

做匀速直线运动的质点在 $t_0$ 时的位置为 $x_0$， 速度为 $v_0$， 且加速度始终等于常数 $a_0$， 求任意时刻的速度和加速度 $x(t)$．

我们首先由\autoref{VnA1_eq8}~\upref{VnA1} 得到速度函数为
\begin{equation}
v(t) = v_0 + \int_{t_0}^t a_0 \dd{t'} = v_0 + a_0 (t - t_0)
\end{equation}
然后再次积分得到位置函数
\begin{equation}
x(t) = x_0 + \int_{t_0}^t [v_0 + a_0 (t' - t_0)] \dd{t'} = x_0 + v_0 (t - t_0) +  \frac12 a_0 (t - t_0)^2
\end{equation}
