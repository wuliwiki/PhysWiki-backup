\subsection{安装robomaster python sdk}
Win + R 输入 cmd 
\verb|或|
点开开始菜单，在搜索框中输入 cmd ，在搜索结果中，对着命令提示符程序，单击鼠标右键，菜单中点击选择 以管理员身份运行 

并输入以下指令

\begin{lstlisting}[language=pythonC]
pip install robomaster
\end{lstlisting}

如果网络较差，多次都安装失败，可以尝试用镜像qing'hua
