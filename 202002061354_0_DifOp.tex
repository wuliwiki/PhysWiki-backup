% 微分算符

我们可以更形式化一些， 将算符与被算符作用的函数分离开并用一个符号表示， 例如另
\begin{equation}
\Q A = \qty(\dv{x} + 1)
\end{equation}
\begin{equation}
\Q A f(x) = \qty(\dv{x} + 1)f(x) = \dv
\end{equation}

\begin{equation}
\qty(\dv{x} + 1)^2 = \dv[2]{x} + 2\dv{x} + 1
\end{equation}

\begin{equation}
\qty(\pdv{x} + \pdv{y})^2 = \pdv[2]{x} + \pdv[2]{y} + 2\pdv{}{x}{y}
\end{equation}
