Singularity 类似于 Docker\upref{Docker}，多用于科学计算（尤其是计算机集群）。因为 docker 的运行需要管理员（root）权限，它在共享的计算机集群中使用时可能会让普通用户获得 root 权限，从而导致安全问题。
