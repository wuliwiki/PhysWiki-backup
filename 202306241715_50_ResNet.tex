% 残差网络
% keys 残差 网络 卷积网络 变体

\textbf{残差网络}（Resdual network, ResNet）是卷积神经网络的一种常见的变体。与原始的卷积网络所不同的是，残差网络学习的是源数据到源数据与标签之间的残差的映射。假设一个残差网络的输入张量为$\bvec x$，输出张量为$\bvec y$，残差网络学习的是$F: \bvec x \rightarrow \bvec y-x$。数学表达式为：
\begin{equation}
\bvec y=F(\bvec  x;\bvec  w)+\bvec x
\end{equation}
其中，$\bvec x$和$\bvec y$分别为输入和输出张量，$\bvec w$为权值。
