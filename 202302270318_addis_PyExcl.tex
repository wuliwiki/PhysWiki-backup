\subsection{CSV}
CSV 是一个简单的表格格式， 用文本编码。
\begin{lstlisting}[language=python]
import csv
with open('test.csv', newline='') as csvfile:
    # sheet 的类型是 _csv.reader
    sheet = csv.reader(csvfile)
    for row in sheet: # row 的类型是 list， 元素都是 str
        print(', '.join(row))
\end{lstlisting}
