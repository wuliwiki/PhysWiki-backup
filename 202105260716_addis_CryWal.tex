\subsection{安全事项}
\begin{itemize}
\item 建议使用不易中病毒的操作系统生成钱包（如 iOS， Mac， Linux 等）， 如果你使用 windows， 确保你的系统和软件都是正版的， 没有录屏软件， 并先进行杀毒．
\item 强烈不建议使用第三方输入法（在 iOS 上， 即使使用第三方输入法， 也不要打开全部权限）
\end{itemize}
