\pentry{拉格朗日方程（额外广义力）\upref{LagEqQ}}

在拉格朗日方程（\autoref{LagEqQ_eq1}~\upref{LagEqQ}）
\begin{equation}
\dv{t} \pdv{L}{\dot q_i} = \pdv{L}{q_i} + Q_i^{(e)}
\qquad (i=1,\dots,N)
\end{equation}
中， 若 $L$ 不直接含时， 且系统不随时间变化（即每个广义坐标 $q_i$ 不随时间变化）， 那么所有的广义力恒为零
\begin{equation}
Q_i = -\pdv{V}{q_i} + Q_i^{(e)} = 0 \qquad (i = 1,\dots,N)
\end{equation}
