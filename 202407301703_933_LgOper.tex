这篇文章会介绍大型运算符，他们往往代表了一个过程。

\subsection{累积过程}
\begin{itemize}
\item 求和符号：$\sum$
\item 求积符号：$\prod$
\item 上积符号：$\coprod$
\item 并集符号：$\bigcup$
\item 交集符号：$\bigcap$
\item 析取符号：$\bigvee$
\item 合取符号：$\bigwedge$
\item 积分符号：$\int$
\end{itemize}
