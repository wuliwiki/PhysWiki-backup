$$\psi(x, 0) = \sqrt{\frac{8}{5a}} (1 + \cos \frac{\pi x}{a}) \sin \frac{\pi x}{a} \quad (0 \leq x \leq a)~$$ 
(1)求后来的某--时刻$t_0$ 的波函数;\\
(2)求在$t=t_0$ 时刻的平均能量;\\
(3)求在$t=t_0$时在势阱左半部($0\leq  x \leq \frac{a}{2}$) 发现粒子的率。30分
