\subsection{快捷键}
\begin{itemize}
\item Mac 的 Delete 键是 Win 的 backspace
\item Mac 的 Cmd 是 Win 的 Win 键
\item Mac 的 Opt 是 Win 的 Alt 键
\item \verb|Cmd + c|， \verb|Cmd + v| 复制粘贴
\item \verb|Cmd + c| 然后 \verb|Cmd + Alt + v| 移动
\item \verb|Cmd + Del| 把文件移动到回收站
\item \verb|Cmd + z| 撤销
\end{itemize}
