1．已知程序如下： \\
\begin{lstlisting}[language=cpp]
int S(int n)
{
    return(n<=0)?0：s(n-1)+n；
}

void main()
{ 
    cout<<S(1)；
}
\end{lstlisting}
程序运行时使用栈来保存调用过程的信息，自栈底到栈顶保存的信息依次对应的是 \\
A．main( )→S(1)→S(0) $\quad$ B．S(0)→S(1)→main( ) \\
C．main( )→S(0)→S(1) $\quad$ D．S(1)→S(0)→main( )
