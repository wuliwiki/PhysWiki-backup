递推关系
\begin{equation}
L_{n+1}^\alpha (x) = [(2n + 1 + \alpha  - x)L_n^\alpha (x) - (n + \alpha )L_{n - 1}^\alpha (x)]/(n + 1)
\end{equation}
\begin{equation}
L_0^\alpha (x) = 1
\qquad
L_1^\alpha (x) = 1 + \alpha  - x
\end{equation}  
罗德里格斯公式
\begin{equation}
L_n^\alpha (x) = \frac{x^{-\alpha} \E^x}{n!} \dv[n]{x} (\E^{-x} x^{n+\alpha})
\end{equation}
氢原子中的拉盖尔函数是 $L_n(x) = L_n^0(x)$．
