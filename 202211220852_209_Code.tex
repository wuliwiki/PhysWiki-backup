\subsection{VS code}

VSCode（全称：Visual Studio Code）是一款由微软开发且跨平台的免费源代码编辑器．

优点：开发环境易用、安装插件简单方便、可中文、打开速度快

缺点：代码提示、纠错相对没那么好、运行速度不快

VS code 官方下载地址：https://code.visualstudio.com/
