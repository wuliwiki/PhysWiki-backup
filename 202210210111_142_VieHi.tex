\begin{theorem}{}
考虑上一元二次方程 $a x^2 + b x + c = 0$，其有两个根 $x_1$ 和 $x_2$，则有：
$$\begin{aligned}
x_1 + x_2 &= -\frac{b}{a} \\
x_1 x_2 &= \frac{c}{a}
\end{aligned}$$
\end{theorem}

证明：
$$\begin{aligned}
a x^2 + b x + c &= a (x - x_1) (x - x_2) \\
a x^2 + b x + c &= a (x - x_1) (x - x_2) \\
\end{aligned}$$
