\subsection{什么是插值}
插值法，就是在函数表达式未知的情况下，通过已知点处的函数值来估计未知点处函数值的方法。具体来说，就是：已知函数 $f(x)$ 在 $\[a] $ 上有定义，且已经测得在 $x_0,x_1,\cdots, x_n$ 处的函数值为 $y_0 = f(x_0),y_1 = f(x_1), \cdots, y_n = f(x_n)$, 如果存在一个简单易算的函数 $p(x)$， 使得$$p(x_i) = f(x_i)~, i = 1,2, \cdots, n~
$$则称 $p(x)$ 是 $f(x)$ 的插值函数。

插值法是数值计算中一种常用且基础的方法，在数据挖掘、数学建模等诸多领域都有广泛的应用。
