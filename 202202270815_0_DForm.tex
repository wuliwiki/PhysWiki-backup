考虑二维的高斯定理
\begin{equation}\label{DForm_eq1}
\oint \bvec f \vdot \uvec n \dd{l} = \int \div \bvec f \dd{s}
\end{equation}
其中 $\dd{l}$ 是曲线上的一个小线段， 曲线逆时针为证． $\uvec n$ 是该线段的法向量， 向外为正．
