% LaTeX 简介（除公式）

考虑到许多网友只用过 LaTeX 编辑公式而没有写过正文， 这里来结合本编辑器做一个简要的介绍．

LaTeX 是一种所见非所得的排版语言， 即用户编辑的是代码， 需要经过编译过程才能获得最终的显示效果．

\subsection{环境}
一个完整的 LaTeX 文档是由许多环境构成的， 环境的格式如下
\begin{lstlisting}
\begin{环境名}[可选设置]
...
...
\end{环境名}
\end{lstlisting}
其中 “[可选设置]” 不一定会出现． 在一个完整的 LaTeX 文档中， 最大的环境就是 document 环境．
