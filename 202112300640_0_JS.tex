% JavaScript 笔记

\pentry{HTML 基础}
JavaScript 常见于网页中， 一般浏览器都可以运行． 我们以 Chrome 浏览器为例演示 \verb|Hello World| 程序．

\begin{lstlisting}[language=js]
<!DOCTYPE html>
<html>
	<body>
		<div id = a></div>
		<script>
            console.log("hello world 1!");
            window.alert("hello world 2!");
            document.write("hello world 3!")
			document.getElementById("a").innerHTML = "hello world 4!";
		</script>
	</body>
</html>
\end{lstlisting}
把这段程序复制到一个文本文件， 并命名为 \verb|test.html|， 用 Chrome 打开即可自动运行． 按 F12 可打开调试窗口和命令行．
