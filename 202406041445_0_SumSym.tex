\begin{equation}
\sum_{i=m}^n a_i = a_m + a_{m+1} + \dots + a_n~.
\end{equation}
许多时候，如果已经提到了求和中 $i$ 取哪些值，就

\begin{equation}
\sum_i C a_i = C\sum_i a_i~.
\end{equation}

\begin{equation}
\sum_i a_i \sum_j b_j = \sum_{i,j} a_i b_j~.
\end{equation}
