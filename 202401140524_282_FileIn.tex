一个储存设备本质上就是一根很长的纸带，划分成许多小格，每个格子可以读或写上 0 或 1。 并且每个读写都可以指定读或写第几个格子。这就是著名的图灵机中的储存模型。

这个模型看起来很简单，但在实际中我们通常对储存设备有一些性能上的要求：
\begin{itemize}
\item 纸带足够长，可以容纳需要保存的数据
\item 读写速度快
\item 写入的数据长时间不会自然损坏
\item 有防误写的机制，例如可以查看纸带在之前某个时间点的数据
\end{itemize}
