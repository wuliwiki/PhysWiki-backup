\begin{definition}{数论函数}
以自然数集或正整数集为值域的函数称为数论函数．
\end{definition}

下面列出常见的数论函数:
\begin{example}{简单数论函数}
\begin{itemize}
\item 单位函数$I(n)$
I(n)=
\begin{cases}
1, n=1\\
0, n>1.
\end{cases}
\item 恒等函数$u(n)$
\[u(n)=n\]
\end{itemize}

\end{example}
