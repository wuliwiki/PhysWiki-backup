（4）一维无限深方势井
$$V(x) = 
\begin{cases} 
0 & 0 < x \leq a \\
\infty & x < 0, x > a 
\end{cases}~$$
中，微观粒子能量的本征值为
$$E_n = \frac{h^2 \pi^2 n^2}{2ma^2} \quad (n = 1, 2, 3, \ldots)~$$

$$\psi_n(x) = \sqrt{\frac{2}{a}} \sin \left( \frac{n \pi }{a} x \right)~$$

$$\psi(x) = \frac{4}{\sqrt{a}} \sin \left( \frac{\pi x}{a} \right) \cos^2 \left( \frac{\pi x}{a} \right)~$$
