% Python 注释
% keys Python|注释

\begin{itemize}
\item 注释可用于解释 Python 代码．
\item 确保对模块, 函数, 方法和行内注释使用正确的风格．
\item 注释可用于提高代码的可读性．
\end{itemize}

\subsection{单行注释}
Python 中单行注释以 # 开头，例如：
\begin{lstlisting}[language=python]
# 这是一个注释，注意标准格式要在#后面加一个空格再加上注释
print("你好，世界！")  # 这是一个注释，注意在代码wei'bu'd
\end{lstlisting}
