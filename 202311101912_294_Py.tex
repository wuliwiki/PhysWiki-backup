\subsection{标识符}

\begin{enumerate}
\item 标识符是指在使用Python的过程中定义名字时的一种规范，主要包括以下几种：
\begin{itemize}
\item 开头不可以是数字
\item 必须由数字，字母，下划线组成
\item 区分大小写
\item 不能使用内置关键词（比如：print,int,float等等）
\end{itemize}
