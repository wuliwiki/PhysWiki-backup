\begin{itemize}
\item \verb`sudo ifconfig` 显示网卡信息, 以及 ip 地址 (inet addr)
\item \verb`sudo hpclient` 显示 ip 地址
\item \verb`sudo /etc/init.d/networking restart` 重启网络服务 
\item 要从某个网址下载文件, 只需安装 wget 软件即可, 使用方法如 \verb`wget http://...`
文件会下载到当前文件夹
\end{itemize}
