\textbf{艾里函数（Airy Function）}是微分方程
\begin{equation}
y'' - xy = 0
\end{equation}
两个线性无关解， 分别记为 $\opn{Ai}(x)$ 和 $\opn{Bi}(x)$．


\subsection{变形}
令 $a, b\in \mathbb R$， 那么
\begin{equation}
y'' + (ax + b) y = 0
\end{equation}
的通解是
\begin{equation}\label{AiryF_eq1}
y(x) = C_1\opn{Ai}\qty(\frac{-ax-b}{\abs{a}^{2/3}}) + C_2 \opn{Bi}\qty(\frac{-ax-b}{\abs{a}^{2/3}})
\end{equation}
