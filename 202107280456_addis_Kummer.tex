% Kummer 函数（1F1）

Kummer 函数 $M(a, b, z)$， 也叫合流超几何函数 $_1F_1(a, b, z)$， 是以下微分方程的解
\begin{equation}
z\dv[2]{f}{z} + (b-z)\dv{f}{z} - a f = 0
\end{equation}

Kummer 函数是超几何函数\upref{HypGeo}的一个特例， 
\begin{equation}
{_pF_q}(a_1,\dots, a_p; b_1, \dots, b_q; z) = \sum_{n=0}^\infty \frac{(a_1)_n\dots (a_p)_n}{(b_1)_n\dots(b_q)_n} \frac{z^n}{n!}
\end{equation}

\begin{equation}
M(a, b, z) = \sum_{s = 0}^\infty \frac{(a)_s}{1}
\end{equation}
