\subsection{要求}

\begin{figure}[ht]
\centering

\caption{两个最小单元，每个都支持链接} \label{fig_thanks_1}

\begin{itemize}
\item 感谢墙和百科
\item 同一个用户可以根据捐款数额获取多个单元，可以连通也可以不连通。
\item 每个用户平均每个单元的代码不能超过 1000 字节。
\item 每个单元允许修改 1 次。
\item 原始模板中可以提供一些公共的 \verb`<style>`， 后来的用户也可以使用其他用户定义的 \verb`<style>`.
\end{itemize}
