\begin{itemize}
\item 广泛用于科学计算： 高能物理、天文、生物信息
\item 由 HDF Group 开发
\item 支持文件夹结构，多维数组，图片，表格，文字，支持大数据量，支持数据压缩，支持小部分读取
\item 支持 C, C++, Fortran, Java, Python, MATLAB
\item HDFView 是一个 GUI 程序可以查看 HDF5 文件。
\end{itemize}
